LoadJobConfiguration
in package
  • Application
implements JobConfigurationInterface

LoadJobConfiguration.php : 37

Represents a configuration for a load job. For more information on the available settings please see the [Jobs configuration API documentation](https://cloud.google.com/bigquery/docs/reference/rest/v2/jobs#configuration).

Example:

use Google\Cloud\BigQuery\BigQueryClient;

$bigQuery = new BigQueryClient();
$table = $bigQuery->dataset('my_dataset')
    ->table('my_table');
$loadJobConfig = $table->load(fopen('/path/to/my/data.csv', 'r'));
