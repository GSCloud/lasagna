DataTransferServiceGapicClient.php : 90

Service Description: The Google BigQuery Data Transfer Service API enables BigQuery users to configure the transfer of their data from other Google Products into BigQuery.

This service contains methods that are end user exposed. It backs up the frontend.

This class provides the ability to make remote calls to the backing service through method calls that map to API methods. Sample code to get started:

$dataTransferServiceClient = new DataTransferServiceClient();
try {
    $formattedName = $dataTransferServiceClient->projectDataSourceName('[PROJECT]', '[DATA_SOURCE]');
    $response = $dataTransferServiceClient->getDataSource($formattedName);
} finally {
    $dataTransferServiceClient->close();
}

Many parameters require resource names to be formatted in a particular way. To assist with these names, this class includes a format method for each type of name, and additionally a parseName method to extract the individual identifiers contained within formatted names that are returned by the API.

public checkValidCreds(string $name[, array<string|int, mixed> $optionalArgs = [] ]) : CheckValidCredsResponse
